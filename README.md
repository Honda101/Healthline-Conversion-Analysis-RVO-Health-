# Healthline-Conversion-Analysis-RVO-Health-
Python code used to clean data for RVO Health Take home test

Project Goal: Increasing revenue efficiency for a company called Healthline using strategic analysis.

Tech Stack: Python (Pandas/NumPy) and Tableau.

Key Findings: Just using data from 100,000 page views we uncovered potential revenue gains of $3,486 and a 'Revenue Lift' of 6.79%.
